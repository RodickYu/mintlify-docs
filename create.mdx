## 为什么要共建？

Nodeloc Wiki 致力于为社区成员提供**实用、准确、易懂**的功能说明和操作指南。  
它不是一个人的作品，而是所有 Nodeloc 用户的共同结晶。  
无论是修正文档错误、补充使用技巧，还是分享自己的经验，你的参与都会让 Wiki 更好。

---

## 如何参与

<AccordionGroup>
  <Accordion icon="code-branch" title="通过 GitHub 提交 PR">
    1. 访问 [Nodeloc Wiki GitHub 仓库](https://github.com/nodeloc/mintlify-docs)
    2. Fork 仓库到你的账户
    3. 修改或新增内容（可用 Markdown/MDX 格式）
    4. 提交 Pull Request，并**在描述中说明修改原因和内容**
    5. 等待维护者审核并合并
  </Accordion>

  <Accordion icon="shield-check" title="贡献规则">
    - 内容需确保真实可靠，不包含误导性信息  
    - 请遵守社区规范和法律法规  
    - 避免抄袭其他平台的受版权保护内容  
    - 格式请保持与现有文档一致（标题层级、代码块、表格等）
  </Accordion>
</AccordionGroup>
