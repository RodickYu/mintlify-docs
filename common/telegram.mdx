## Telegram 关联使用说明

通过 Telegram 关联，您可以在 Nodeloc 论坛接收即时通知，方便及时了解社区动态。

---

### 第 1 步：获取 Telegram ID

1. 打开 Telegram，联系机器人 [@nler_bot](https://t.me/nler_bot)。  
2. 机器人会自动返回一个 **唯一 ID**。

![获取 Telegram ID 示例|690x257](upload://e9VzDWzHTUoxoZvFnhbZVjMXqmO.jpeg)

---

### 第 2 步：填写 Telegram ID

1. 登录 Nodeloc 论坛。  
2. 进入 **偏好设置 → 个人资料 → Telegram 通知**。  
3. 将机器人返回的 **ID** 填写到对应输入框中并保存。

![填写 Telegram ID 示例|594x499](upload://x7smHp0q6JybOiRKRpjEevJSZ2i.png)

---

### 第 3 步：使用通知功能

- 填写完成后，您即可通过 Telegram 接收论坛通知，如私信、点赞或重要公告。  
- 如果未收到通知，请检查 ID 是否正确或尝试重新绑定。

<Note>
确保您已开启 Telegram 的消息通知权限，以便及时接收论坛通知。
</Note>
